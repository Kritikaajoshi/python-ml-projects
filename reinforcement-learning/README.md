# Reinforcement Learning with Gymnasium

Implemented RL agents in the CartPole-v1 environment
using the agent-environment interaction loop.

## What's Implemented
- Task 1: Random policy, 100 episodes, reports max/avg/std of rewards
- Task 2: Deterministic toggle policy, alternates left/right each timestep
- Task 3: Video recording per episode using RecordVideo wrapper
- Task 4: Custom environment with random and deterministic policies

## Key Concepts
- Agent-environment loop (state, action, reward)
- Uniform random policy vs deterministic policy
- Episode statistics using RecordEpisodeStatistics wrapper

## Tech
Python, Gymnasium
